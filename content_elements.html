HTML content elements tutorial

Welcome to this tutorial on HTML content elements.

1. Headings

HTML provides six level of headings , <h1> heading element to <h6> heading element, with <h1> being highest level and <h6> being the lowest.

    
    <h1> Heading level 1 </h1>
    <h2> Heading level 2 </h2>
    <h3> Heading level 3 </h3>
    <h4> Heading level 4 </h4>
    <h5> Heading level 5 </h5>
    <h6> Heading level 6 </h6>

2. Paragraphs and Line breaks

Paragraphs are created using the <p> element.
To force a line break within paragraph, use the <br> element.

7. Code and Keyboard Input

The <code> element is used to represent the inline code, while the <kbd> element is used to represent keyboard input
